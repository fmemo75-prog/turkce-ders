01 What is the Accusative Case?

Theory

In Turkish, when a verb has a direct object — the thing that receives the action — you usually need to add a special suffix to that noun. This suffix is called the accusative case (İsmin İ Hali / Yükleme Hali).

The key distinction: use the accusative only for specific, definite objects — something the speaker and listener both know about. If the object is indefinite (any random one), you leave it bare (no suffix).

When to use the accusative

Add the accusative suffix when the object is specific / definite — "the book", "that coffee", "my sister". Do NOT add it for indefinite objects — "a book", "coffee" (in general).

Kitabı okudum → I read the book (specific)
Kitap okudum → I read a book (indefinite)

Two important notes:
1. If the noun ends in a vowel, insert a buffer -y- before the suffix: araba → arabayı.
2. Some consonants change when a suffix is added: ç → c, k → ğ, p → b, t → d (consonant softening). Example: kitap → kitabı, çocuk → çocuğu.
